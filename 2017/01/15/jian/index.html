4、创建+切换分支

创建分支的同时切换到该分支上，命令如下：

git checkout -b [branch name]
1
1
git checkout -b [branch name] 的效果相当于以下两步操作：

git branch [branch name]
git checkout [branch name]

5、将新分支推送到github

命令如下：

git push origin [branch name]
1

6、删除本地分支

命令如下：

git branch -d [branch name]
1
1
例如：

git branch -d gh-dev
1
1
7、删除github远程分支

命令如下：

git push origin :[branch name]
1
1
分支名前的冒号代表删除。
例如：

git push origin :gh-dev
